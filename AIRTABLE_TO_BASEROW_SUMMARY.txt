================================================================================
  KEY DIFFERENCES
================================================================================

LIBRARY:
  Airtable: pyairtable (SDK)
  Baserow:  requests (HTTP)

AUTHENTICATION:
  Airtable: API Token
  Baserow:  API Token (same concept)

TABLE ACCESS:
  Airtable: By name (e.g., 'Fix it')
  Baserow:  By numeric ID (e.g., 456)

PAGINATION:
  Airtable: Automatic (.all() method)
  Baserow:  Manual (page parameter)

FILTERING:
  Airtable: Formula language (like Excel)
  Baserow:  Field-based or client-side

ERROR HANDLING:
  Airtable: Exceptions
  Baserow:  HTTP status codes
